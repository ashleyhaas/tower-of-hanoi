# Tower of Hanoi

Create a web page that allows the user to play the [Tower of Hanoi](https://en.wikipedia.org/wiki/Tower_of_Hanoi) game.

![tower of hanoi gif](/300px-Tower_of_Hanoi_4.gif)

The user should be able to click on towers and move the disks between the towers. The rules are simple: you can only move one disk at a time, and you cannot place a larger disk on a smaller disk. Your game should have 3 towers and 4 disks. The user has won the game once they get all disks on to the right-most tower. Your game should display a message to the user once they have won.

Requirements:
- 3 towers and 4 disks are present on the page
- User can move disks between towers
- User cannot place larger disks on smaller disks
- After the user wins a message is displayed to the user.
